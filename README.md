## 📌 Overview

LUKAS AI is an intelligent desktop voice assistant developed using Python. The project is designed to perform smart automation tasks, respond to voice commands, open applications, search the web, interact with APIs, and provide a user-friendly graphical interface.

The assistant combines voice recognition, natural language processing, automation tools, and GUI development into a single AI-powered desktop application.

This project was developed as a practical implementation of AI-based automation and desktop assistant technologies.

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/LUKAS-AI.git
```

## 2️⃣ Move into Project Folder

```bash
cd LUKAS-AI
```

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

## 4️⃣ Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / MacOS

```bash
source venv/bin/activate
```

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

# 🧠 How LUKAS AI Works

1. User gives a voice or text command.
2. Speech recognition converts voice into text.
3. Command handler processes the instruction.
4. AI engine determines the response/action.
5. Automation module executes the requested task.
6. Assistant responds through voice and GUI.
